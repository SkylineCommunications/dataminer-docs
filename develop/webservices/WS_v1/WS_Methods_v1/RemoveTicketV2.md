# RemoveTicketV2

Use this method to remove a specific ticket.

## Input

| Item       | Format  | Description                                           |
|------------|---------|-------------------------------------------------------|
| connection | String  | The connection ID. See [ConnectApp](xref:ConnectApp). |
| ticketID   | Integer | The ID of the ticket.                                 |

## Output

None.
