# Script

Runs another automation script from within the current script.

```xml
<Exe id="2" type="script">
   <Script>Attach to Ticket</Script>
   <Param type="DEFER">FALSE</Param>
   <Param type="CHECKSETS">TRUE</Param>
   <Param type="OPTIONS">11</Param>
</Exe>
```
