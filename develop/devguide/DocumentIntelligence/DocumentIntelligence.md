# Document Intelligence

Document Intelligence provides an API for analyzing documents using the [DataMiner Assistant DxM](xref:Assistant_DxM). You can send one or more documents with custom instructions to extract or analyze content using an LLM.

The main entry point is the [DocumentIntelligenceHelper](xref:DocIntel_DocumentIntelligenceHelper) class, which exposes the `AnalyzeDocuments` method.
