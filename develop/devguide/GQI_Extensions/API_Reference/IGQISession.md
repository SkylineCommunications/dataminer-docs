# IGQISession interface

## Definition

- Namespace: `Skyline.DataMiner.Analytics.GenericInterface`
- Assembly: `SLAnalyticsTypes.dll`

This interface provides properties regarding the GQI session that invoked the extension.

Available from DataMiner 10.6.3/10.5.0 CU12/10.6.0 CU0 onwards.<!-- RN 44509 -->

## Properties

### Username

The username of the user that executed the GQI querry. This value is equivalent to the DomainUserName property found on the IConnection object.
