# BrokerGateway element

Specifies whether the BrokerGateway-managed NATS solution (nats-server service) is used instead of the SLNet-managed NATS solution (NAS and NATS services).

This element is set to "true" when the system has been [migrated to BrokerGateway-managed NATS](xref:BrokerGateway_Migration). From DataMiner 10.6.0/10.6.1 onwards, this element is no longer used.

## Content Type

boolean
