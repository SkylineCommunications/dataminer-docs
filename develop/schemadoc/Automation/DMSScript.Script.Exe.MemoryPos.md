# MemoryPos element

Specifies the memory position to get or set.

## Type

string

## Parent

[Exe](xref:DMSScript.Script.Exe)

## Remarks

Format "memory file ID:position of the item to retrieve or set" e.g., "2:100".

## Examples

```xml
<MemoryPos>2:4</MemoryPos>
```
