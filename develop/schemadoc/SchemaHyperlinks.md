# Hyperlinks schema

DataMiner Hyperlinks XML schema.

For more information about hyperlinks, refer to [Hyperlinks.xml](xref:Hyperlinks_xml).

## Root element

[HyperLinks](xref:HyperLinks)

## Example

```xml
<HyperLink id="100"
        version="2"
        name="Web page"
        menu="root"
        type="reference"
        alarmColumn="true">
  10
</HyperLink>
```
