# Service profile benchmarks

## Specifications of the test servers

### Metrics 1 - 17

- Intel Xeon Silver 4210
- 8 cores (16TH) VM
- 32 GB RAM
- SSD (NVMe)
- Windows Server 2019 Standard

### Metrics 18 -21

- Intel Core i7-8700 CPU \@3.20 GHz
- 6 cores
- 32 GB RAM
- SSD
- Cassandra & Elasticsearch
- Windows 10 Pro
