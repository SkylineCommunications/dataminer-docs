# Verify BrokerGateway Migration

Verifies whether the DataMiner System has migrated from the SLNet-managed NATS solution to the BrokerGateway-managed NATS solution.

From DataMiner 10.6.0/10.6.1 onwards<!--RN 43861 -->, this prerequisite is available by default and runs automatically when you upgrade.

Migrating to BrokerGateway is possible from DataMiner 10.5.0 [CU2]/10.5.5 onwards. For details, refer to [Migrating to BrokerGateway](xref:BrokerGateway_Migration).

> [!NOTE]
> To ensure early rollout at Skyline, this prerequisite is enabled from 10.5.10 onwards for Skyline employees with a DataMiner Agent in the skyline.local domain.
