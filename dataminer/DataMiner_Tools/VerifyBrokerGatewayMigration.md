# Verify BrokerGateway Migration

Verifies whether the DataMiner System has migrated from the SLNet-managed NATS solution to the BrokerGateway-managed NATS solution.

From DataMiner 10.6.0/10.6.1 onwards<!--RN 43861 -->, this prerequisite is available by default and runs automatically when you upgrade.

For more information about the migration to BrokerGateway, refer to [Migrating to BrokerGateway](xref:BrokerGateway_Migration).

> [!NOTE]
> For Skyline employees that have a DataMiner agent in the skyline.local domain, this prerequisite is enabled from 10.5.10 onwards. 
> Migration to BrokerGateway can be done from 10.5.5 onwards following the [Migrating to BrokerGateway](xref:BrokerGateway_Migration) procedure.
