# Assistant chat

The Assistant chat component embeds the [DataMiner Assistant](xref:DataMinerAssistant) directly in a dashboard or low-code app, so users can start a conversation in context without leaving the page.

![Assistant chat](~/dataminer/images/AssistantChat.png)<br>*Assistant chat component in DataMiner*

## Configuration options

In the component settings, you can configure the following:

- **Title**: The title shown at the top of the component.

- **Subtitle**: Additional text shown below the title.

- **Agent**: The assistant agent that will handle the conversation.

## Component action (low-code apps)

In low-code apps, this component exposes the **Start new conversation** component action. This lets you start a new assistant conversation through an event or another component action, without requiring users to interact with the Assistant chat component itself.
