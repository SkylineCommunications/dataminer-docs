# Starting or stopping a DataMiner node using DataMiner Taskbar Utility

1. On the DataMiner Agent you want to start or stop, right-click the [DataMiner Taskbar Utility](xref:DataMiner_Taskbar_Utility) system tray icon.

1. Depending on what you wish to do, select either *Stop DataMiner*, *Start DataMiner* or *Restart DataMiner*.

1. In the confirmation window, enter your username and a message if necessary, and click *Yes*.
