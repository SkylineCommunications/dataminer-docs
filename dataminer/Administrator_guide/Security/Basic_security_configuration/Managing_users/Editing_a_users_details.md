# Editing a user's details

To edit a user's details:

1. In DataMiner Cube, go to System Center > *Users / Groups* > *users*, and select the user whose information you want to edit.

   Alternatively, you can also open the user card for the user in question. See [Viewing user information](xref:Viewing_user_information).

1. Fill in the new data in the applicable fields.

   As domain users are managed by the domain, most details for such users cannot be adjusted via Cube. For more details, see [Types of users](xref:Types_of_users).

   To change the password for the built-in [Administrator user](xref:User_management#default-users), you must be logged in with the Administrator user account.

1. Click the *Apply* button in the lower-right corner.

> [!NOTE]
> In order to quickly change your own password, while you are logged in with your account, right-click your name in the *Users / Groups* section of the System Center module, and select *Change password*.
