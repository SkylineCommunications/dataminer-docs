# Linking shapes to DataMiner items

Shapes in Visio can be linked to a multitude of DataMiner items, ranging from DataMiner elements or views to specific commands or files, webpages, scripts and more.

You can link a shape to:

- [an element, a service or a redundancy group](xref:Linking_a_shape_to_an_element_a_service_or_a_redundancy_group)

- [a view](xref:Linking_a_shape_to_a_view)

- [an element parameter](xref:Linking_a_shape_to_an_element_parameter)

- [an alarm](xref:Linking_a_shape_to_an_alarm)

- [a history alarm](xref:Linking_a_shape_to_a_history_alarm)

- [an alarm filter](xref:Linking_a_shape_to_an_alarm_filter)

- [the alarm state of a Data Display page](xref:Linking_a_shape_to_the_alarm_state_of_a_Data_Display_page)

- [a DLL file](xref:Linking_a_shape_to_a_DLL_file)

- [a WFM component](xref:Linking_a_shape_to_a_WFM_component)

- [a MULTIPLE SET command](xref:Linking_a_shape_to_a_MULTIPLE_SET_command)

- [a program or a file](xref:Linking_a_shape_to_a_program_or_a_file)

- [a SET command](xref:Linking_a_shape_to_a_SET_command)

- [a video thumbnail](xref:Linking_a_shape_to_a_video_thumbnail)

- [a webpage](xref:Linking_a_shape_to_a_webpage)

- [an Automation script](xref:Linking_a_shape_to_an_Automation_script)

- [a matrix parameter](xref:Linking_a_shape_to_a_matrix_parameter)

- [a calculation involving multiple parameters](xref:Linking_a_shape_to_a_calculation_involving_multiple_parameters)

- [a trend component](xref:Linking_a_shape_to_a_trend_component)

- [an aggregation rule](xref:Linking_a_shape_to_an_aggregation_rule)

- [an Asset Manager table](xref:Linking_a_shape_to_an_Asset_Manager_table)

- [a dashboard component](xref:Linking_a_shape_to_a_dashboard_component)

- [an EPM object](xref:Linking_a_shape_to_an_EPM_object)
