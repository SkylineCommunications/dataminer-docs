# Hiltron HCS4 - LNB

This connector is used by DVEs (Dynamic Virtual Elements) created by the parent connector [Hiltron HCS4](xref:Connector_help_Hiltron_HCS4).

This connector can be used to monitor a LNB module contained in a Hiltron HCS4 controller.

## How to use

This element using this connector is automatically generated by a parent element using the **Hiltron HCS4** driver. It displays information about an LNB module managed by the Hiltron HCS4 unit controller.
