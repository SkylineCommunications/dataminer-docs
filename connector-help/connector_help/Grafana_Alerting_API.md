## Configuration

### Connections

#### HTTP Connection

This connector uses an HTTP connection and requires the following input during element creation:

HTTP CONNECTION:

- **IP address/host**: The polling IP or URL of the destination.
- **IP port**: The IP port of the destination (default: *3000*).

### Initialization

On the **General** page, specify a Bearer token with the **Bearer Token** parameter.

## How to use

This connector contains an alerts table.
