# SatOps 1.3.0 - Preview

> [!IMPORTANT]
> We are still working on this release. Release notes may still be modified, added, or moved to a later release. Check back soon for updates!

> [!NOTE]
> This version requires:
>
> - DataMiner 10.5.11/10.6.0 or higher, as well as DataMiner Web 10.6.2 or higher.
> - [MediaOps Plan 1.5.1](xref:MediaOps_Plan_1.5.1) or higher.

## New features

#### Antenna Inventory: Arc of steerable antennas now taken into account [ID 44954]

It will now no longer be possible to book an antenna for a satellite that is outside its arc. For this purpose, every antenna that is controlled by an Antenna Control Unit will now have a configured antenna arc.
