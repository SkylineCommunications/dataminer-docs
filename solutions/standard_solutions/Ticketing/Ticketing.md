# Ticketing

The DataMiner Ticketing Standard Solution provides a way to create, manage, and track tickets within the DataMiner ecosystem. It is designed to support operational workflows and seamless integrations with both internal DataMiner components as with external ticketing systems.

The solution is inspired by the **TM Forum TMF 621 Trouble Ticket Management specification**, meaning that only a subset of this specification is currently supported. Additional functionality can however be added when required by a specific project.
