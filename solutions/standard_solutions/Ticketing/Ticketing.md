# Ticketing

The DataMiner Ticketing Standard Solution provides a way to create, manage, and track tickets within the DataMiner ecosystem. It is designed to support operational workflows and seamless integrations with both internal DataMiner components and external ticketing systems.

The Ticketing Solution includes support for the following use cases:

- Manual ticket creation and tracking.
- Alarm-driven ticket creation.
- Linking tickets to elements, alarms, and assets.
- Cross-application navigation within DataMiner.
- Integration with external ticketing systems.

The solution includes a [dedicated app](xref:TicketingArchitecture#ticketing-frontend) where you can view and manage tickets.

> [!NOTE]
> The solution is inspired by the **TM Forum TMF 621 Trouble Ticket Management specification**, meaning that only a subset of this specification is currently supported. However, additional functionality can be added when required for a specific project.
