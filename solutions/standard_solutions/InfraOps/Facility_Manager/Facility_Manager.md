# Facility Manager

With the Facility Manager application, you can create a hierarchical view of your infrastructure. In this overview, you can define and manage buildings, floors, rooms, zones, rows, desks, and racks.

![Facility Manager Overview](~/solutions/images/Facility_Manager_Overview.png)

## App Overview

- ![Overview](~/solutions/images/FacilityManager_Overview.png) **Overview**: Provides quick insight in facilities, rack units, and power consumption.

- ![Facilities](~/solutions/images/FacilityManager_Facilities.png) **[Facilities](xref:Facilities_Page)**: Allows you to manage the facilities in your infrastructure.

- ![Rooms](~/solutions/images/FacilityManager_Rooms.png) **[Rooms](xref:Rooms_Page)**: Allows you to manage the available rooms.

- ![Racks](~/solutions/images/FacilityManager_Racks.png) **[Racks](xref:Racks_Page)**: Allows you to manage the available racks.

- ![Room Designer](~/solutions/images/FacilityManager_Room_Designer.png) **Room Designer**: Allows you to design a room in detail, including the placement of different racks and zones.

- ![About](~/solutions/images/FacilityManager_About.png) **About**: Provides information about the version of the app and allows you to configure the Google Maps API Key used for the maps displayed in the app.
