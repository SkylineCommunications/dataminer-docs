# Cable Types

Here the user can define the type of cables available when establishing data/network connections between assets. Same as the Device Types, the solution already comes with a predefined list that the user can configure and expand:

![Cable Types side panel](~/solutions/images/Asset_Manager_Cable_Types_Side_Panel.png)

When creating or updating a Cable Type, the user must provide a unique name. Additionally, they can specify a description for the cable and assign category tags to help organize and classify cables effectively:

![Cable Type Wizard](~/solutions/images/Asset_Manager_Cable_Type_Wizard.png)
