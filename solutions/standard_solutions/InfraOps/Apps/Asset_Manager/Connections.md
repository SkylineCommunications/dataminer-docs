# Connections

Page with an Overview of all connections created. From here the user can also create Power and Data connections (same as in Asset Details).

![Connections Page](~/solutions/images/Asset_Manager_Connections_Page.png)

On the top right, by clicking the option “Connections Panels”, users will have access to the Connection Panel View that provides a very useful visual aid that highlights the connections established between assets that have the device type “Connection Panel” and “regular” Assets:

![Connection Panel View](~/solutions/images/Asset_Manager_Connection_Panel_View.png)
